# Next.js + React (TypeScript) quick tutorial

This short tutorial is targeted at developers coming from Vue and Laravel.

- Start the dev server:

```bash
npm install
npm run dev
```

- Open: http://localhost:3000/tutorial

What you'll see in the `app/tutorial` page:

- `PropsExample` — shows how to pass props (like Vue props).
- `Counter` — shows local state via `useState` (like Vue data).
- `FormExample` — shows controlled inputs (equivalent to `v-model`).

Notes:

- Files added: `app/tutorial/page.tsx` and client components in `app/tutorial/components`.
- The page is a client component to keep examples simple and interactive.
